Governed memory has a lifecycle

Approve

Check consent, source, sensitivity, and scope.

Retrieve

Apply identity, policy, relevance, and freshness filters.

Use

Inject bounded context with provenance into the agent run.

Correct

Accept user edits, contradictions, and improved evidence.

Expire

Supersede, archive, or delete stale records.

Audit

Trace writes, reads, omissions, policy decisions, and deletions.

Memory Contract

Typed, scoped, inspectable, and reversible.
