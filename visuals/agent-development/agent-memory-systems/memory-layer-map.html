Agent memory is a layered data system

Model runtime

Current prompt, tool calls, reasoning budget, and temporary scratch context.

owned by harness/runtime
Session state

Thread history, checkpoints, resumable workflow state, and recent raw turns.

owned by runtime
Long-term memory

Distilled facts, episodes, preferences, procedures, and account observations.

owned by memory contract
Product truth

Authoritative customers, contracts, tickets, orders, policy records, and permissions.

owned by application
Retrieval indexes

Vector, text, graph, recency, and entity indexes that make memory discoverable.

owned by data platform
Governance

Consent, access control, retention, deletion, correction, audit, and eval datasets.

owned by product and security
