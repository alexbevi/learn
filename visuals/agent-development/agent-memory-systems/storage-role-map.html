Storage is not one bucket called memory

Product truth

Customers, contracts, orders, tickets, entitlements, and policy.

Needs: authority and constraints

Memory records

Typed observations, facts, episodes, procedures, scopes, provenance.

Needs: flexible schema

Vector index

Semantic recall over memories, transcripts, documents, and summaries.

Needs: relevance search

Graph index

Entities, relationships, temporal edges, ownership, and multi-hop context.

Needs: inspectable links

Cache

Semantic cache, prompt fragments, recent retrievals, and latency-critical context.

Needs: speed and expiry

Audit and evals

Traces, memory writes, selected memories, corrections, deletions, datasets.

Needs: accountability
