Memory is split by purpose, scope, and retrieval path

1

Thread history

Conversation continuity for a specific ticket or session.

Store: message/thread records
2

Working memory

Stable facts the agent should keep available while working with a resource.

Store: structured state
3

Semantic recall

Vector-backed retrieval over prior memories and context.

Store: vector index + embedder
4

Observational memory

Compression-oriented observations and reflections for long-running context.

Store: distilled observations
