MongoDB operations cross three boundaries before bytes persist or return

Client command insertMany(), aggregate(), read/write concern, sessions, transactions
MongoDB server execution parsing, validation, authorization, locks, planner, SBE/classic execution, storage cursor API
WiredTiger engine transactions, row-store B-trees, cache, eviction, reconciliation, history store, checkpoints, log
Operating system and media filesystem cache, compressed pages, data files, journal files, fsync latency, block device behavior

Read surface
